## 🎯 About Arino

**Arino** is a modern digital agency platform showcasing professional web development, mobile app development, and UI/UX design services. Built with a sleek dark theme and bilingual support (English/Arabic).

**أرينو** هو منصة وكالة رقمية حديثة تعرض خدمات تطوير الويب والتطبيقات وتصميم واجهات المستخدم بشكل احترافي.

## 🚀 Getting Started

```bash
# Clone repository
git clone https://github.com/its3li/Arino.git
cd Arino

# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```
